Githubの使い方．
Terminalを起動し，cloneしたいファイルの置き場所まで移動する．
Git clone URL
でcloneする．ファイルを新たに追加した際は，

1: git add ファイル名
2: git commit  -> vi が起動して，適当な位置で i を押してINSERTモードに変える．コミットメッセージを書いて, esq押してINSERTモードを抜ける．:wq(保存して終了)を押してviを終了する．

3: git push

で変更内容等をリモートリポジトリ上に反映させる．

リモートリポジトリ上で変更があった場合は，

git pull

で変更をローカルに反映させる．
